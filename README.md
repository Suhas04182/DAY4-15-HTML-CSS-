# Day 15 - Testimonial Card Project

## Description
This Day 15 project is a simple **Testimonial Card** built using HTML and CSS.

A testimonial card is commonly used on websites to display customer reviews, feedback, or user experiences in an attractive format.

## Files Included
- `index.html` → Main webpage structure
- `style.css` → Styling for the testimonial card
- `README.md` → Project documentation

## Features
- Clean testimonial card design
- Circular user profile image
- User name and role section
- Customer review text
- Hover animation effect
- Beginner-friendly code

## Technologies Used
- HTML5
- CSS3

## Learning Goals
By building this project, you will learn:

1. How to create card-based layouts
2. How to style profile images
3. How to use box shadows
4. How to apply hover effects
5. How to structure testimonial sections

## Concepts Covered
- CSS Border Radius
- Box Shadow
- Hover Effects
- Text Alignment
- Card Design
- Responsive Layout Basics

## How to Run
1. Download or clone the repository
2. Open the project folder
3. Open `index.html` in your browser
